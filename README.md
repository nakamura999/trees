## サイト概要
ヘアスタイルなどの情報配信。
コミュニケーションが取れる。
### サイトテーマ
美容師の情報発信。

### テーマを選んだ理由
自分が元美容師だったので、美容師にとって役にたつツールを作りたいと思った為。

### ターゲットユーザ
美容師。すべての人。髪の毛で悩んでる人など。

### 主な利用シーン
美容師の情報発信。掲示板などを利用して相談ができる。
